# Add Pattern to Network Tool

A clean Minecraft mod template built with Architectury framework for multi-platform development.

## Requirements

### Forge
- **Minecraft**: 1.20.1
- **Forge**: 47.3.0+
- **Architectury API**: Required
- **ClothConfig**: Required

### Fabric
- **Minecraft**: 1.20.1
- **Fabric Loader**: Latest
- **Architectury API**: Required
- **ClothConfig**: Required

## Development

This project uses Gradle with Architectury for multi-platform mod development.

### Build Commands
- `./gradlew build` - Build all modules
- `./gradlew :forge:runClient` - Run Minecraft client for testing
- `./gradlew clean` - Clean build artifacts
